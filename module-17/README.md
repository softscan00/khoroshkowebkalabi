## 🎯 Цель модуля

Изучить основы HTML — языка разметки веб-страниц.

---

## 📚 Теория

### Что такое HTML?

**HTML** (HyperText Markup Language) — язык разметки, который определяет структуру веб-страницы. HTML использует **теги** для описания элементов страницы.

### Структура HTML-документа

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Заголовок страницы</title>
</head>
<body>
  <!-- Содержимое страницы -->
</body>
</html>
```

- `<head>` — метаданные (не видны на странице)
- `<body>` — видимое содержимое

### Основные теги

```html
<h1>Заголовок</h1>          <!-- Заголовки h1-h6 -->
<h2>Подзаголовок</h2>

<p>Параграф текста.</p>      <!-- Текст -->

<a href="https://example.com">Ссылка</a>  <!-- Ссылки -->

<img src="photo.jpg" alt="Описание">      <!-- Изображения -->

<div class="card">Контейнер</div>          <!-- Блочный контейнер -->
```

### Атрибуты

Атрибуты добавляют дополнительную информацию к элементам:

```html
<img src="photo.jpg" alt="Описание">
<div id="header" class="container">Контент</div>
```

- `src` — путь к файлу (для `<img>`)
- `alt` — описание изображения (для `<img>`)
- `class` — класс для стилизации
- `id` — уникальный идентификатор

## 💡 Примеры

### Пример: Карточка товара

```html
<div class="product-card">
  <img src="phone.jpg" alt="Смартфон">
  <h2>Смартфон XYZ</h2>
  <p>Цена: 29 990 ₽</p>
</div>
```

Обрати внимание на структуру: контейнер `<div>` с классом, внутри — изображение, заголовок и текст. Задание построено по тому же принципу.

## ✏️ Задание

### Описание

Напишите функцию `createHTML`, которая **возвращает строку** с HTML-разметкой карточки пользователя.

Вы не создаёте HTML-файл — вы пишете JavaScript-функцию, которая собирает HTML как обычную строку и возвращает её через `return`.

```javascript
// Пример идеи — функция возвращает строку с HTML:
function example() {
  const color = 'red';
  return `<div class="${color}">Текст</div>`;
}

example(); // '<div class="red">Текст</div>'
```

### Требования

1. Функция принимает объект пользователя: `{ name, email, avatar }`
2. Возвращает **строку** с HTML-разметкой (через шаблонный литерал)
3. Структура карточки:
   - Контейнер `<div class="user-card">`
   - Изображение `<img>` с src из avatar и alt с именем
   - Заголовок `<h2>` с именем
   - Параграф `<p>` с email

### Пример результата

```javascript
createHTML({ name: 'Иван', email: 'ivan@mail.ru', avatar: 'photo.jpg' })
```

Должен вернуть:
```html
<div class="user-card">
  <img src="photo.jpg" alt="Иван">
  <h2>Иван</h2>
  <p>ivan@mail.ru</p>
</div>
```

## 💭 Подсказки

<details>
<summary>Подсказка 1: Направление</summary>
Какой синтаксис в JavaScript позволяет удобно вставлять значения переменных прямо внутрь строки? Подумайте о шаблонных строках.
</details>

<details>
<summary>Подсказка 2: Структура</summary>

Функция должна возвращать одну строку, содержащую HTML-разметку. Самый удобный способ собрать строку с подставленными значениями — шаблонные литералы (обратные кавычки), где через `${выражение}` можно вставить любое свойство объекта прямо внутрь строки.

</details>

<details>
<summary>Подсказка 3: Подход</summary>

Создайте функцию, которая принимает объект пользователя и возвращает шаблонный литерал с HTML-разметкой. Внутри литерала постройте структуру: корневой `div` с классом `user-card`, внутри него `img` (используйте свойства `avatar` для `src` и `name` для `alt`), затем `h2` с именем и `p` с email. Подставляйте свойства объекта через `${}`. Обратите внимание на точные отступы — тесты могут проверять формат строки.

</details>
